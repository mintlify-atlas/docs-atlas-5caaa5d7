# Get Started with Sable

This guide will help you get up and running with Sable in just a few minutes.

## Using the Web App

The fastest way to start using Sable is through the web application. No downloads required!

<Steps>
  <Step title="Access Sable">
    Navigate to [app.sable.moe](https://app.sable.moe/) in your web browser.
  </Step>
  
  <Step title="Log in to your Matrix account">
    Enter your Matrix credentials:
    - **Homeserver**: Your Matrix homeserver (e.g., matrix.org, sable.moe)
    - **Username**: Your Matrix username
    - **Password**: Your Matrix password
    
    <Info>
      Don't have a Matrix account? You can create one at [matrix.org](https://matrix.org/) or any other Matrix homeserver.
    </Info>
  </Step>
  
  <Step title="Explore featured communities">
    After logging in, you'll see featured communities and rooms. Browse through them to find interesting conversations, or skip to start your own.
  </Step>
  
  <Step title="Join a room">
    Click on a room from the featured list, or:
    - Use the search feature to find specific rooms
    - Click the **+** button to join by room address (e.g., `#general:matrix.org`)
    - Accept invitations from your contacts
  </Step>
  
  <Step title="Send your first message">
    Once you've joined a room:
    - Type your message in the message box at the bottom
    - Use Markdown for formatting (bold, italic, code, etc.)
    - Press **Enter** to send
    
    <Tip>
      Press **Shift+Enter** to add a new line without sending the message.
    </Tip>
  </Step>
</Steps>

## Using the Desktop App

Prefer a native desktop experience? Download and install the Sable desktop app.

<Steps>
  <Step title="Download the desktop app">
    Visit the [releases page](https://github.com/7w1/sable/releases/latest) and download the installer for your operating system:
    - **Windows**: Download the `.exe` installer
    - **Linux**: Download the AppImage or distribution-specific package
  </Step>
  
  <Step title="Install and launch">
    **Windows**: Run the installer and follow the setup wizard.
    
    **Linux**: Make the AppImage executable and run it:
    ```bash
    chmod +x Sable-*.AppImage
    ./Sable-*.AppImage
    ```
  </Step>
  
  <Step title="Log in">
    Enter your Matrix credentials just like in the web app. The desktop app will sync your account and preferences.
  </Step>
</Steps>

</Note>
