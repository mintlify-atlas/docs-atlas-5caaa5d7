## Web Application

The simplest way to use Sable is through the web application. No downloads or installation required.

**Access URL:** [app.sable.moe](https://app.sable.moe/)

### Features
- Always up-to-date with the latest stable features
- Works on any modern web browser
- Cross-platform (Windows, macOS, Linux, mobile)
- No storage space required

### Browser Requirements
- Modern browsers with ES2020+ support
- Recommended: Chrome, Firefox, Safari, Edge (latest versions)

<Note>
  The web app is updated frequently as soon as features are deemed stable, so you'll always have access to the newest improvements.
</Note>

## Desktop Application

For a native desktop experience with system integration, download the Sable desktop application.

### Download

Get the latest version from the [GitHub releases page](https://github.com/7w1/sable/releases/latest).

### Supported Platforms

<CardGroup cols={2}>
  <Card title="Windows" icon="windows">
    Windows installer available for all recent Windows versions
  </Card>
  
  <Card title="Linux" icon="linux">
    AppImage and distribution-specific packages available
  </Card>
</CardGroup>

### Installation Steps

<Steps>
  <Step title="Download">
    Navigate to the [releases page](https://github.com/7w1/sable/releases/latest) and download the appropriate installer for your operating system.
  </Step>
  
  <Step title="Install">
    **Windows:** Run the `.exe` installer and follow the installation wizard.
    
    **Linux:** Download the AppImage, make it executable (`chmod +x`), and run it. Alternatively, use your distribution's package manager if a native package is available.
  </Step>
  
  <Step title="Launch">
    Start Sable from your applications menu or desktop shortcut.
  </Step>
</Steps>

### Desktop App Benefits
- Native system notifications
- System tray integration
- Better performance on some systems
- Offline availability
- Desktop file integration

## Progressive Web App (PWA)

You can install Sable as a Progressive Web App directly from your browser for an app-like experience.

### How to Install

1. Visit [app.sable.moe](https://app.sable.moe/) in a supported browser
2. Look for the install prompt in your browser's address bar
3. Click "Install" to add Sable to your device

<Info>
  PWA installation is available on Chrome, Edge, and other Chromium-based browsers on desktop and mobile.
</Info>

## Choosing the Right Option

| Option | Best For | Pros |
|--------|----------|------|
| **Web App** | Quick access, trying Sable | No installation, always updated, cross-platform |
| **Desktop App** | Daily use, power users | Native experience, system integration, offline support |
| **PWA** | Mobile users, minimal installation | App-like experience, works across devices |
| **Self-Hosted** | Organizations, privacy-focused users | Full control, customization, data sovereignty |
