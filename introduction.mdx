# Welcome to Sable

Sable is a Matrix client where you can enjoy the conversation using a simple, elegant and secure interface protected by end-to-end encryption with the power of open source. And colorful cosmetics. Because yes.

Forked from [Cinny](https://github.com/cinnyapp/cinny/), Sable builds upon a solid foundation to deliver enhanced features and user experience improvements.
